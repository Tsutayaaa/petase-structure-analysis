# petase-structure-analysis

Project files for a PETase structure-analysis poster.

## Contents

- `assets/logos/`: logos and externally sourced branding/reference images used in the poster
- `data/`: structure and ligand input files
- `docking/`: docking configuration and output-related files
- `figures/chimerax/`: ChimeraX raw exports and rendered figures used during poster preparation
- `poster/source/`: editable poster source files
- `ramachandran/`: Ramachandran plot scripts and poster figures
- `references/`: literature PDFs cited or referenced in the poster workflow
- `receptor.pdb`: receptor structure used in the workflow

## Notes

This repository stores the project assets and analysis outputs associated with the poster work.
